# java-inventory-system
A Java Inventory Management System (IMS) integrated with MySQL, showcasing my understanding of Java fundamentals, JDBC, SQL queries, and OOP. Product data is stored and managed through MySQL as part of my learning journey in building real-world Java applications.

Features
- Add, delete, and search products
- Persistent storage using MySQL
- Modular design using object-oriented programming
- Input validation and exception handling

Future Features to add:
-update product

Technologies

Java | MySQL | JDBC

Getting Started

Clone the repository.
Set up the MySQL database and configure the connection in the project.
Run the Java application.

Learning Goals
Practice building a database-driven Java application
Apply OOP concepts in a real-world project
Strengthen JDBC and SQL skills
